# 📊 Sales Analytics Dashboard

---

## 📌 Overview

Sales data is more than numbers — it tells a story 📈

This project focuses on analyzing sales data to uncover trends, measure performance, and generate actionable insights.
An interactive dashboard is built to help stakeholders make **data-driven business decisions**.

---

## 🎯 Project Objective

The goal of this project is to:

* Analyze overall sales performance
* Identify top-performing products & regions
* Track revenue trends over time
* Understand customer purchasing behavior
* Support strategic business decisions

---

## 📂 Dataset

**File:** `sales_data.csv`

### 🔑 Key Columns

* Order ID
* Order Date
* Product Name
* Category
* Sales Amount
* Quantity
* Customer Segment
* Region / Location
